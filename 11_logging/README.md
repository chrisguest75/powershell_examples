# README

Demonstrate logging libraries for Powershell

```ps1
Find-Module  -Name "poshlog"    
Import-Module poshlog
Get-InstalledModule
```

```ps1
Install-Module poshlog -Scope CurrentUser 
Import-Module poshlog   
Find-Command -ModuleName "poshlog"
```
